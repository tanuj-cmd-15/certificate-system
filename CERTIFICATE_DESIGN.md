# Certificate Design - NEXUX TANTRA 2K26 Style

## 🎨 Design Overview

The certificate has been redesigned to match the NEXUX TANTRA 2K26 style with professional technical aesthetics suitable for SPPU Department of Technology events.

## ✨ Design Features

### 1. Color Scheme
- **Primary Green:** `#1a5f3f` (Dark forest green)
- **Secondary Green:** `#2d8659` (Medium green)
- **Accent Green:** `#7ec99f` (Light mint green)
- **Background:** White with geometric patterns

### 2. Geometric Elements

**Top Left Corner:**
- Triangular gradient overlay
- Green color gradient from dark to light
- Creates modern, technical feel

**Bottom Left Corner:**
- Triangular green block
- Anchors the design
- Professional appearance

**Hexagonal Pattern:**
- Technical hexagon grid overlay
- Subtle background pattern
- Represents technology and innovation

**Circuit Pattern:**
- Electronic circuit design in bottom right
- Dots and connecting lines
- Emphasizes technical nature

### 3. Header Section

**Left Side:**
- SPPU Logo (circular emblem with "S")
- University branding

**Right Side:**
- DoT (Department of Technology) badge
- Green gradient background
- Modern design

**Title:**
- Large "CERTIFICATE" text
- Green color (#1a5f3f)
- Letter spacing for elegance
- "OF APPRECIATION" subtitle

### 4. Event Badge
- Green rounded badge
- Event name (e.g., NEXUX TANTRA 2K26)
- Prominent display
- White text on green background

### 5. Content Layout

**Presentation Text:**
- "THIS CERTIFICATE IS PROUDLY PRESENTED TO:"
- Uppercase, bold, letter-spaced
- Professional formatting

**Recipient Name:**
- Large, bold text
- Green color
- Top and bottom borders
- Centered display

**Achievement Details:**
- Prize information (First Prize, Second Prize, etc.)
- Category information
- Event details
- University and department name

### 6. Certificate Number Box
- Yellow/gold background (#fff3cd)
- Gold border (#ffc107)
- Monospace font for certificate number
- Easy to read and copy

### 7. Details Grid
- 2x2 grid layout
- Light gray gradient background
- Green left border accent
- Contains:
  - Issue Date

### 8. Footer Section

**Left Side:**
- Date display
- Issue date

**Right Side:**
- Signature line
- "DR. ADITYA ABHYANKAR"
- "HEAD OF DEPARTMENT"
- Professional signature block

### 9. Verification Badge
- Green background with checkmark
- "VERIFIED CERTIFICATE" text
- Authenticity confirmation
- Prominent display

## 📋 Certificate Fields

### Required Fields:
1. **Recipient Name** - Person receiving the certificate
2. **Recipient Email** - For email notification
3. **Event Name** - e.g., NEXUX TANTRA 2K26

### Optional Fields:
4. **Prize** - First Prize, Second Prize, Third Prize, Winner, Runner Up, Participation
5. **Category** - Competition category (Web Development, AI/ML, IoT, etc.)
6. **Issuer Name** - Defaults to SPPU DOT if not specified

### Auto-Generated:
7. **Certificate Number** - SPPU-DOT-YYYYMMDDHHMMSS-XXXX
8. **Issue Date** - Automatic timestamp
9. **Verification URL** - Auto-generated link
10. **LinkedIn Share URL** - Pre-configured sharing link

## 🎯 Usage Example

### Admin Panel Form:
```
Recipient Name: Rahul Sharma
Recipient Email: rahul@example.com
Event Name: NEXUX TANTRA 2K26
Prize: First Prize
Category: Web Development
Issuer Name: (leave blank for default)
```

## 🚀 How to Use

1. **Login as Admin:** http://localhost:8080/login
2. **Go to Admin Panel**
3. **Fill Certificate Form:**
   - Recipient Name
   - Recipient Email
   - Event Name (e.g., NEXUX TANTRA 2K26)
   - Prize (First Prize, Second Prize, etc.)
   - Category (Web Development, AI/ML, etc.)
4. **Click Generate Certificate**
5. **View Certificate:** Click certificate number to verify
6. **Share:** Use LinkedIn button or print
